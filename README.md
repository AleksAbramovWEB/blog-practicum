## Сдача проектной работы четвертого спринта (Создание блога Spring Boot) 

### ✅  Устанавливаем переменное окружение в .env

```
UPLOAD_IMAGE_DIR=

DB_URL=jdbc:postgresql:
DB_USERNAME=
DB_PASSWORD=

TEST_UPLOAD_IMAGE_DIR=

TEST_DB_URL=
TEST_DB_USERNAME=
TEST_DB_PASSWORD=

```

### ✅ Сборка/старт приложения
```bash
chmod +x run.sh
./run.sh
```

### ✅ Запуск тестов
```bash
chmod +x run-test.sh
./run-test.sh
```
